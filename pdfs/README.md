# PDF Training Documents

Place your PDF training documents in this folder.

## Instructions

1. Copy your PDF files into this directory
2. The application will automatically scan this folder for PDF files
3. Click "Reload PDFs" in the UI or restart the backend to load new PDFs
4. Ask questions about the content in your PDFs using the chatbot interface

## Supported Formats

- PDF files (`.pdf` extension)

## Notes

- PDFs are processed when the application starts
- Use the "Reload PDFs" button in the UI to reload documents without restarting
- Large PDFs may take some time to process
